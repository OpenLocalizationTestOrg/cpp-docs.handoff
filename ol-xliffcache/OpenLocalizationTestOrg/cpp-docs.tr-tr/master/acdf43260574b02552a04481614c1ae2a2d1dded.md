# Compiler Error C3866
function call missing argument list  
  
 Inside a nonstatic member function, a destructor or finalizer call did not have an argument list.  
  
 The following sample generates C3866:  
  
```  
// C3866.cpp  
// compile with: /c  
class C {  
   ~C();  
   void f() {  
      this->~C;   // C3866  
      this->~C();   // OK  
   }  
};  
```
