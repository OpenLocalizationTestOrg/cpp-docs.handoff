# Module::ReleaseNotifier::Release
Deletes the current Module::ReleaseNotifier object if the object was constructed with a parameter of `true`.  
  
## Syntax  
  
```cpp  
void Release() throw();  
```  
  
## Requirements  
 **Header:** module.h  
  
 **Namespace:** Microsoft::WRL
