# CAccessorBase::Close
Closes the accessors.  
  
## Syntax  
  
```  
  
void Close( );  
  
```  
  
## Remarks  
 You must call [ReleaseAccessors](../../data/oledb/caccessorbase-releaseaccessors.md) first.  
  
## Requirements  
 **Header:** atldbcli.h
