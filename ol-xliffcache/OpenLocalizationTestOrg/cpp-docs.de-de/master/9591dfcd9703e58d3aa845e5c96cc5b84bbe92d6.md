# Resource Compiler Fatal Error RC1101
no resource binary filename specified  
  
 The Rename Output (/fo) option was not followed by a filename.  
  
 Use the following syntax for the /fo option:  
  
 **RC /fo**   
 ***outfilename infilename.* rc**
