# RAND_MAX
## Syntax  
  
```  
  
#include <stdlib.h>  
  
```  
  
## Remarks  
 The constant `RAND_MAX` is the maximum value that can be returned by the `rand` function. `RAND_MAX` is defined as the value 0x7fff.
