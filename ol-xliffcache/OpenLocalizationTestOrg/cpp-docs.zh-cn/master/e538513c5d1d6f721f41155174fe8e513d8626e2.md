# CAccessorBase::GetNumAccessors
Retrieves the number of accessors created by the class.  
  
## Syntax  
  
```  
  
ULONG GetNumAccessors( ) const;  
  
```  
  
## Return Value  
 The number of accessors created by the class.  
  
## Requirements  
 **Header:** atldbcli.h
